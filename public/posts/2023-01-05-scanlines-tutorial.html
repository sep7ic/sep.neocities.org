i thought id just dump scanline code here. also thought it might be good to make it into a tutorial so youre welcome

    to get a "scanlines" effect, youre gonna want to put this css code with the rest of your css code.

            /*------------scanlines---------------*/
.scanlines {
  position: relative;
  overflow: hidden;
}
.scanlines:before, .scanlines:after {
  display: block;
  pointer-events: none;
  content: '';
  position: absolute;
}
.scanlines:before {
  width: 100%;
  height: 2px;
  z-index: 9999;
  background: rgba(0, 0, 0, 0.3);
  opacity: 0.75;
  animation: scanline 6s linear infinite;
}
.scanlines:after {
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  z-index: 9999;
  background: linear-gradient(to bottom, transparent 50%, rgba(0, 0, 0, 0.3) 51%);
  background-size: 100% 4px;
  animation: scanlines 1s steps(60) infinite;
}
/* ANIMATE UNIQUE SCANLINE */
@keyframes scanline {
  0% {
    transform: translate3d(0, 200000%, 0);
  }
}
@keyframes scanlines {
  0% {
    background-position: 0 50%;
  }
}

    after, pick the element you want to have the scanlines effect on and put class="scanlines" right after the element tag. thats basically it!

    you can reference the scanlines class in the body tag but im pretty sure the page becomes unusable (havent tested) so id recommend referencing the tag in your main div.
